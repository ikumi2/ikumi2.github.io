本地配置

下载
  1. 下载git和nodejs并安装(全默认即可)
  2. 在D盘里新建blog文件夹
  3. 在blog文件夹右键Git Bash Here
安装

  1. 安装hexo

  2. 初始化hexo

  3. 安装git部署

  4. 安装文章管理

  5. 安装搜索(可选)

  6. 安装live2d插件(可选)

  7. 安装网站地图插件(可选)

配置

  1. 添加个人信息

  2. 创建密钥

  3. 完成后使用 hexo s 启动本地服务 访问 http://localhost:4000 即可看到你的网站

服务器配置

安装
  • 安装nginx和git
    CentOS Debian(Ubuntu)
配置nginx

  1. 编辑nginx配置文件

  2. 更改网站根目录为 /home/git/www

  3. 保存退出后重启nginx

添加用户

  1. 添加git用户

  2. 进入git用户

添加密钥

  1. 创建.ssh目录

  2. 进入.ssh目录创建authorized_keys文件

  3. 用记事本打开本地的 id_rsa.pub 把公钥复制到 authorized_keys 中

  4. 编辑好后测试密钥是否登录成功git用户，成功后继续下一步

添加git仓库

  1. 添加网站根目录文件夹和git仓库

  2. 在 git 仓库的 hooks 里创建 post-receive 文件 并给予执行权限

  3. 设置钩子自动部署

  4. 以上完成服务器配置 接着下一步的部署

github域名绑定
